# 使用方法

1. nodeをinstall
2. git clone reveal.jsのリポジトリ
3. $ npm install
4. $ npm startでlocalhost:8000で起動
5. vscodeをinstall

---

# 使用方法

1. vscode-revealをinstall
2. test.mdを作成
3. command + p 「reveal」でreveal js presentation by sideを実行  
4. markddown編集画面とプレビュー画面の2窓で作業
